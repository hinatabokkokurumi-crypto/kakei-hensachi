# 家計偏差値

## 概要
都道府県・世帯構成・年代・年収別にあなたの家計を同じ属性の世帯と比較するサイトです。
「我が家の支出、多いのか少ないのか？」を数秒で診断できます。

## 機能
- 🏠 **世帯情報の入力**
  - 都道府県、世帯人数、年代、年収を選択
  
- 📊 **支出比較**
  - 同属性世帯の平均値と比較
  - 食費、光熱費、交通費など項目別に表示
  
- 📈 **可視化**
  - グラフで一目瞭然
